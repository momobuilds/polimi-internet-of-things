# Homework 01: Parking Occupancy Node

This homework explores an ESP32-based parking occupancy node using an HC-SR04 ultrasonic sensor, ESP-NOW communication, and deep-sleep power management.

## Contents

- `notebooks/parking-occupancy-node-analysis.ipynb`: main notebook with system description, code discussion, and energy analysis.
- `reports/parking-occupancy-node-report.pdf`: exported report version of the notebook.
- `reports/challenge-description.pdf`: assignment statement.
- `reports/energy-plot.pdf`: supporting energy visualization.
- `data/`: measurement CSV files used in the analysis.
- `assets/`: supporting images and setup figures.
