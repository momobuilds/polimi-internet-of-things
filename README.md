# Internet of Things Coursework

Coursework repository for the Internet of Things course at Politecnico di Milano.

This repository contains three homework submissions covering embedded sensing, protocol analysis, and LoRa network simulation.

## Repository Structure

- `homework-01-parking-occupancy-node/`: ESP32 parking occupancy node with ESP-NOW communication and energy analysis.
- `homework-02-coap-mqtt-analysis/`: CoAP, MQTT, and MQTT-SN traffic analysis with protocol-level exercises.
- `homework-03-lora-network-simulation/`: LoRa challenge material and LoRaSim-based reproduction of selected paper figures.
- `archive/raw-exports/`: Original zip exports and duplicate PDF bundles kept locally for reference and excluded from Git.

## Notes

- Most analytical work is documented in Jupyter notebooks and exported PDF reports.
- Some notebooks were originally prepared for Google Colab; paths may need small adjustments before re-running locally.
